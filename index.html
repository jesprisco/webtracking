Instructions

To host a site of your own, you must do the following:

  1. Fork this repo
  2. Go to forked repo and click add file
  3. Name file [brandid].html
  4. Edit file and paste in the webtracking sample page code found in the View Instructions button for your brand under Settings-> Webtracking
  5. Do a find and replace for 'openInformzAlertDialog' and replace with 'alert'
  6. Commit changes
  7. Edit the index.html page (this page) and create a link under Pages to your new site
  8. Commit changes
  9. Create a Pull Request from your forked branch to the main branch and let Jesiah know

Note:

For Dev2 you must host the web_trk/sp.js file manually. I have hosted it on this site. RC will point to cdn.informzrc.net to grab that file instead. We can either change the sample code to point to the informzrc.net cdn or paste the same code as is and it will point to the web_trk file hosted here. Keep in mind when changes are made to this file we will need to manually update it.
